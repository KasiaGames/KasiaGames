Why a drawing robot?

    We choose to make a drawing robot because we wanted to know how to control motors using sensors. We also wanted to learn how a microcontroller can communicate with a pc.

    Problems while making the robot.

    While we where making the drawing robot we encountered some problems. One of those problems was that we couldnt get the stepper motors to work using the A4988 motor driver. After trying for multiple times to get the A4988 to work we decided that it was better to use a differnt motor driver. The motor driver we are using now is the L298N motor driver which is used to power 2 dc motors or one stepper motor. A benifit of this driver is that it can connect to two lego motors. We also have a problem where the distance sensor doesnt give the correct value. We havent fixed that problem yet.
